# 1. ゲーム概要
 * プレイ人数: 2〜4人
 * プレイ時間: 30分
 * 勝利条件: ゲーム終了時、最も高い勝利点（VP）を獲得したプレイヤーの勝利。

# 2. コンポーネント
 * マップボード: 14の拠点（集/島）とルートが描かれた盤面。
 * カード: 全65枚（文化C / 産業I / 政治P / 特殊）。
 * 資源トークン: 5種（F食糧 / O鉱石 / G交易品 / M機械 / D公文書）。
 * プレイヤーコマ: 各色1個。

# 3. ゲームの準備
 * マップ: 全員、コマを西のスタート地点**「01. 天神集 マカチ」**に置く。
 * 資源: 全員、初期資源として F(食糧)×2、O(鉱石)×1 を受け取る。
 * 手札: 山札をよく切り、各プレイヤーに 4枚 ずつ配る。
 * 山札: 残りのカードは裏向きで山札とする。

# 4. ラウンドの流れ
全員同時進行で、以下の4.1 ~ 4.9を順に行う。

## 4.1. 緊急配給（救済）
 * 手番開始時、手持ち資源がひとつもない（0個の）プレイヤーは、政府から即座に**【F 食糧】を1個**受け取る。

## 4.2. 計画フェイズ
 * 手札からカードを1枚選び、伏せて出す（「移動」か「予約」か決める）。

## 4.3. 実行フェイズ
一斉にカードを表にする。
## 4.4. 【A：移動】
   * カード左上の数字（No.）分だけコマを進める。
   * Wはワープ（マップ上の好きなマスへ移動）。
   * 周回: マップはループしている。終点（14）を過ぎたら始点（01）へ戻って進む。
   * 島への移動: ストックへ資源1個を通行料として支払う必要がある（払えないなら入れない）。
   * ※カード効果で免除されている場合は無料。
## 4.5. 【B：予約】
   * カードを予約エリアに置く。コマは移動しない。
## 4.6. マス効果フェイズ
 * 現在いるマスの効果を処理する（予約の人も発動）。

| マス   | 効果                                                               |
| ------ | ------------------------------------------------------------------ |
| 資源   | 山札を1枚めくる。数字分得る。(0=任意1個、1〜3=その数、W=3個) |
| 交易所 | 手持ち2個 → 好きな資源1個。                                        |
| 山札   | 山札から1枚引く。                                                  |
| 首都   | 好きな資源1個。                                                    |

## 4.7. 混雑フェイズ（インタラクション）
同じマスに2人以上のコマがある場合、以下を行う。
 * 判定: そのマスにいるプレイヤー同士で**「達成エリアのカード枚数」**を比べる。
   * 枚数が少ない側が**「受け取る側」**。
   * 枚数が多い側が**「渡す側」**。
 * 徴収:
   * 「渡す側」全員は、自分の手持ち資源から1つを選び、場（プール）に出す。
   * ※資源がない人は免除。
 * 分配:
   * プールされた資源を、**「受け取る側」の人数で山分け（等分）**する。
   * 割り切れなかった端数（余り）は、ストックに戻す（破棄）。
## 4.8. 達成フェイズ
 * 資源を支払い、予約カードを達成エリアへ移動。
 * 効果は即座に有効。
## 4.9. 補充フェイズ
 * 手札を4枚まで補充。
 * 
# 5. ゲーム終了と得点計算
 * 誰かが 合計6枚 達成したラウンド終了時に終了。
 * 予約ペナルティ: 予約エリアに残ったカードのVP（右上の数字）をマイナスする。
