# 仮称：アイルの街道 (AIL LIME)

## 1. ゲーム概要

* プレイ人数: 2〜4人
* プレイ時間: 30~45分
* 勝利条件: ゲーム終了時、最も高い勝利点（VP）を獲得したプレイヤーの勝利。

## 2. コンポーネント

* マップボード: 14の拠点（集/島）と周回ルート。
* カード: 全65枚（C/I/P 各14枚 + N 18枚 + SP 5枚）。
* 資源トークン: 5種。（F食糧 / O鉱石 / G交易品 / M機械 / D公文書）
* プレイヤーコマ: 各色1個。

## 3. ゲームの準備

* マップ: 全員、コマをスタート地点「01. マカティ」に置く。
* 資源: 全員、初期資源として F(食糧)×2、O(鉱石)×1 を受け取る。
* 手札: 山札をよく切り、4枚ずつ配る。
* 親: スタートプレイヤーを決める。

---

## 4. ラウンドの流れ

### 4.2. 計画フェイズ

* 手札から1枚選び、伏せて出す。
* 選択: 移動か予約か決める。（上下で示す）

### 4.3. 実行フェイズ

全員一斉にカードを表にします。その後、**親マーカーを持つプレイヤーから時計回りに**、1人ずつ以下の手順で手番を行います。

#### 4.3.1. 移動

カード左上の「数字（No.）」の数だけ、時計回りにコマを進めます。

1. **移動の実行:**
   
    - **通常移動:** 数字の分だけ進む。分岐は任意選択。
   
    - **ボーナス移動:** 「2+F」などの表記がある場合、移動後に**即座にその資源をストックから得る**。

2. **捨て札:** 使用したカードは捨て札にする。

#### 4.3.2. マス効果フェイズ（生産）

現在いるマスの効果を処理する。

#### 4.3.3. 混雑判定

移動終了時、そのマスに**他のプレイヤー**がいるかを確認する。 

- **ほかのプレイヤーがいる場合:** そのマスに入ったプレイヤーは、手持ちの資源1個を選び、**そのマスの上に**置く。
- **ほかのプレイヤーがいない場合:** そのマスに置かれた全ての資源を獲得する。
- 資源を支払い、「予約エリア」にあるカードを達成する。

### 4.5. 補充フェイズ

* カードを1枚引く。
* 親マーカーを隣に移す。

---

## 5. ゲーム終了と得点計算

* 終了条件: 2回目の山札切れでフラグ。そのラウンド＋ラスト1周で終了。

* 得点: 達成VP + ボーナスVP - 予約ペナルティ
