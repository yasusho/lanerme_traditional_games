# 仮称：アイルの街道 (AIL LIME)

## 1. ゲーム概要

* プレイ人数: 2〜4人
* プレイ時間: 45〜60分
* 勝利条件: ゲーム終了時、最も高い勝利点（VP）を獲得したプレイヤーの勝利。

## 2. コンポーネント

* マップボード: 14の拠点（集/島）と周回ルート。
* カード: 全65枚（C/I/P 各14枚 + N 18枚 + SP 5枚）。
* 資源トークン: 5種。（F食糧 / O鉱石 / G交易品 / M機械 / D公文書）
* プレイヤーコマ: 各色1個。
* 親マーカー: 1個。

## 3. ゲームの準備

* マップ: 全員、コマを西のスタート地点「01. 天神集 マカチ」に置く。
* 資源: 全員、初期資源として F(食糧)×2、O(鉱石)×1 を受け取る。
* 手札: 山札をよく切り、4枚 ずつ配る。
* 市場: 山札から3枚公開して並べる。
* 親: スタートプレイヤーを決める。

---

## 4. ラウンドの流れ

全員同時進行。4.7（補充）のみ親から。

### 4.1. 緊急配給

手番開始時、資源0個なら、即座に【F 食糧】を1個受け取る。

### 4.2. 計画フェイズ

* 手札から1枚選び、伏せて出す。
* 選択: 移動か予約か決める。（上下で示す）

### 4.3. 実行フェイズ（移動）

一斉にカードを表にする。

【A：移動を選ぶ場合】

* カード左上の数字（No.）分だけコマを進める。
* W: ワープ。（複数人いる場合は親から近いプレーヤー）
* 島への移動: ストックへ資源1個を通行料として支払う。
* 出したカードは捨て札にする。

【B：予約を選ぶ場合】

* カードを予約エリアに置く。移動なし。

### 4.4. マス効果フェイズ（生産）

現在いるマスの効果を処理する。

| マス  | 効果                                 |
| --- | ---------------------------------- |
| 資源  | 山札の一番上を1枚めくり、0:好きな資源を1つ、1-3:その数、W: |
| 交易所 | 手持ち3個 → 好きな資源1個。                   |
| 山札  | 山札（または市場）から1枚引く。                   |
| 首都  | 好きな資源1個。                           |

### 4.5. 混雑フェイズ（インタラクション）

移動先に複数人が同時にいる場合、全員資源を1つそこに置く。

### 4.6. 達成フェイズ

* 資源を支払い、「予約エリア」にあるカードを達成する。

### 4.7. 補充フェイズ（市場ドラフト）

* 親から順に、手札を1枚補充（山札 or 市場）。
* 親マーカーを左隣へ渡す。

---

## 5. ゲーム終了と得点計算

* 終了条件: 3回目の山札切れでフラグ。そのラウンド＋ラスト1周で終了。

* 得点: 達成VP + ボーナスVP - 予約ペナルティ
